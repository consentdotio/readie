# Acme Toolkit

[![npm](https://img.shields.io/npm/v/acme-toolkit.svg)](https://www.npmjs.com/package/acme-toolkit)

A TypeScript toolkit for building internal tools.

## Table of Contents

- [Key Features](#key-features)
- [Installation](#installation)
- [Usage](#usage)
- [Support](#support)

## Key Features

- Typed APIs
- Fast local setup
- Works in Node.js and Bun

## Installation

```bash
npm install acme-toolkit
```

## Usage

1. Import the toolkit and initialize your client.

```ts
import { createClient } from "acme-toolkit";

const client = createClient();
```
